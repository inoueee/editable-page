# Editable Page Toggle (Chrome Extension)

ページ内のテキストを自由に編集できるシンプルなChrome拡張です。
ワンクリックまたは右クリックから、任意のWebページを編集可能に切り替えられます。

---

## ⚙️ 主な機能

### 🖊 編集モード切り替え

* ツールバーアイコンをクリックでON/OFF
* 右クリックメニューからも切り替え可能

### 📌 視覚フィードバック

* 編集モード中はページ上部に固定バーを表示（「編集モード中」）
* フォーカス中の要素を青枠で強調表示

### 🔔 トースト通知

* ON/OFF切り替え時に右下へスライド表示
* 自然なアニメーション付き

### 🧠 状態に応じたメニュー表示

* 右クリックメニューが動的に変化

  * OFF時：「編集モードON」
  * ON時：「編集モードOFF」

---

## 📦 インストール方法

1. このリポジトリをクローンまたはダウンロード

2. Chromeで以下を開く

   ```
   chrome://extensions/
   ```

3. 右上の「デベロッパーモード」をON

4. 「パッケージ化されていない拡張機能を読み込む」をクリック

5. 本リポジトリのフォルダを選択

---

## 🚀 使い方

### 方法①：ツールバーから

* 拡張機能アイコンをクリック → 編集モード切り替え

### 方法②：右クリックから

* ページ上で右クリック → 「編集モードON / OFF」

---

## ⚠️ 注意事項

* 一部のサイト（例：ログインページやSPA）では正常に動作しない場合があります
* 編集内容は保存されません（リロードで元に戻ります）
* セキュリティ上、機密情報の編集には注意してください

---

## 🛠 技術概要

* Manifest V3
* `chrome.scripting` API によるスクリプト注入
* `contenteditable` 属性による編集機能
* `contextMenus` API による右クリック操作
